# root.vpn — Инструкция для клиента (через что и как запускать конфиги)

**🌐 [English](USAGE.md) · Русский**

Каждому пользователю root.vpn выдаёт **два профиля**:

1. **AmneziaWG (UDP/443)** — быстрый, основной. Пробуйте его первым.
2. **VLESS + REALITY (TCP/443)** — запасной для сетей, где режут UDP или делают
   глубокую инспекцию TLS. Используйте, если первый не подключается.

> **Переключение сейчас ручное:** пробуете AmneziaWG; если не подключается
> (UDP заблокирован/режется) — переключаетесь на VLESS. Вариант «одно приложение
> с авто‑фолбэком» (Clash.Meta/Mihomo) — в конце.

---

## Где лежат конфиги (на сервере)

После `sudo awg2 add <name>` сервер печатает пути. Скопируйте их на устройство
(`scp`) или просто отсканируйте QR из вывода терминала.

| Профиль | Файлы (`<name>` — имя клиента) |
|---|---|
| AmneziaWG | `/root/awg/<name>.vpnuri` (ссылка vpn://) · `/root/awg/<name>.vpnuri.png` (QR) · `/root/awg/<name>.conf` · `/root/awg/<name>.png` (QR для .conf) |
| VLESS+REALITY | `/etc/rootvpn/xray/clients/<name>/vless.txt` (ссылка `vless://…`) |

Забрать всё для одного клиента на ПК:

```bash
scp -r root@SERVER:/root/awg/<name>.* .
scp root@SERVER:/etc/rootvpn/xray/clients/<name>/vless.txt .
```

## Приложение и шаги по платформам

### 🪟 Windows
- **AmneziaWG** → **AmneziaVPN** (amnezia.org). Открыть → `+` → *Импорт
  конфигурации* → вставить `vpn://…` (из `<name>.vpnuri`) или загрузить `.conf` →
  Подключиться.
- **VLESS** → **v2rayN** (github.com/2dust/v2rayN, со встроенным Xray). Скопировать
  ссылку `vless://…` → *Серверы → Импорт из буфера* → выбрать → *Сделать активным*
  → включить *Системный прокси* (или *Tun*).

### 🍎 macOS
- **AmneziaWG** → **AmneziaVPN** (App Store / amnezia.org). Импорт `vpn://`.
- **VLESS** → **Hiddify** (App Store) или **Streisand**/**FoXray** (без аккаунта).
  *Добавить профиль → из буфера* (ссылка `vless://`) → Подключить.

### 🤖 Android
- **AmneziaWG** → приложение **AmneziaWG** (Play, `org.amnezia.awg`) — сканировать
  `<name>.png` (QR для .conf); либо **AmneziaVPN** — импорт `vpn://`.
- **VLESS** → **Hiddify** (Play) или **v2rayNG** (APK с GitHub). `+` → *Импорт из
  буфера / Скан QR* → ссылка `vless://` → подключить.

### 📱 iOS
- **AmneziaWG** → **AmneziaVPN** (App Store). Импорт `vpn://`.
- **VLESS** → **FoXray** (бесплатно, полный REALITY/Vision) или **Streisand**
  (бесплатно) или **Shadowrocket** (платно). Добавить из буфера → `vless://`.

### 🐧 Linux
- **AmneziaWG** → headless: `awg-quick up ./<name>.conf` (нужен `amneziawg-tools`),
  либо GUI **AmneziaVPN**.
- **VLESS** → **Hiddify** (AppImage) / **NekoRay** / `xray run -c client.json` /
  **mihomo**. Импорт ссылки `vless://`.

## 🔑 Какой QR/ссылку куда

| Что отсканировали/импортировали | Куда | Результат |
|---|---|---|
| `vpn://` (`.vpnuri` / `.vpnuri.png`) | приложение **AmneziaVPN** | ✅ AmneziaWG |
| `.conf` / его `.png` | приложение **AmneziaWG** или `awg-quick` | ✅ AmneziaWG |
| `vless://` | v2rayN / Hiddify / NekoBox / FoXray / sing‑box / mihomo | ✅ TCP‑фолбэк |

Приложению **AmneziaVPN** нужен формат **`vpn://`** (сырой `.conf` QR оно не
импортирует). Отдельное приложение **AmneziaWG** читает QR от **`.conf`**.

## 🩺 Если не работает (по убыванию частоты)

| Симптом | Причина | Решение |
|---|---|---|
| AmneziaWG не подключается на мобильном/части Wi‑Fi | в сети заблокирован UDP | Переключиться на профиль **VLESS** (TCP/443) |
| VLESS подключился и отвалился через секунды (РФ) | ТСПУ режет Vision на :443 | На сервере: `TCP_TRANSPORT="xhttp"`, затем `sudo awg2` (пере‑экспорт ссылок) |
| Хендшейк прошёл, трафика нет (любой протокол) | IP/ASN сервера режут на уровне сети | Сменить VPS на чистый по репутации / другой регион |
| «Не та» страница / пусто | устаревший конфиг после ротации ключа/SNI | Переимпортировать свежую ссылку (после `awg2 rotate-*`) |
| Подключается, но сайты не открываются | DNS / расхождение часов | Проверить время устройства; туннель отдаёт DNS 1.1.1.1 |
| Импортировали `vpn://` не в то приложение | не то приложение | `vpn://` → AmneziaVPN, `vless://` → Xray‑клиент |

## 🧪 Проверка что трафик реально в туннеле (нет утечек)

После подключения откройте **https://ifconfig.co** — должен показать **IP
сервера**, а не ваш реальный. Затем **https://ipleak.net** /
**browserleaks.com/dns**: DNS‑резолвер и IP — серверные, **IPv6 нет**, локального
DNS не видно.

## ⌨️ Командная строка: меню, языки и флаги

Запуск `sudo awg2` **без команды в терминале** открывает интерактивное меню
(установка / добавить / список / статус / ротация / удаление) — просто нажмите
**Enter** для установки. Установка через `curl … | sudo bash` остаётся полностью
неинтерактивной.

Глобальные флаги (работают с любой командой и с одной‑строчной установкой):

| Флаг | Действие |
|---|---|
| `--lang=en\|ru\|zh\|vi` | язык интерфейса (также берётся из `$LANG`; есть и env `RVLANG=`) |
| `--quiet` / `-q` | только предупреждения/ошибки — удобно для скриптов/cron |
| `--verbose` | подробный вывод (показывает точную команду upstream и т.п.) |
| `--ascii` | чистый ASCII, без цвета и рамок (учитывает и `NO_COLOR`) |

```bash
sudo awg2 --lang=ru                 # интерактивное меню на русском
sudo awg2 --lang=zh add alice       # добавить клиента, вывод на китайском
curl -fsSL https://raw.githubusercontent.com/antidetect/root.vpn/main/install.sh \
  | sudo RVLANG=ru REALITY_DEST=dl.google.com bash   # установка, русский UI
```

Выбранный язык и конфигурация сохраняются между перезагрузками, которые может
запросить базовый инсталлятор — просто повторите ту же команду, чтобы продолжить.

## ⭐ Опционально: одно приложение + авто‑фолбэк (продвинуто)

Чтобы не жонглировать двумя профилями, клиент **Clash.Meta / Mihomo** может
держать **обе** точки (AmneziaWG и VLESS) в одной подписке и сам выбирать
рабочую:

- **Windows / macOS / Linux:** **Clash Verge Rev**
- **Android:** **FlClash** (или ClashMetaForAndroid)
- **iOS:** проверенного единого AWG+VLESS варианта пока нет — оставайтесь на двух приложениях.

Для этого нужна небольшая **подписка**, генерируемая на сервере (Mihomo YAML с
прокси `amnezia-wg` + `vless`/REALITY, объединёнными группой `fallback`). Это в
плане работ (`awg2 sub`) — см. [DESIGN‑v2](DESIGN-v2-tcp-masking.md). Пока —
приложения по платформам выше.
